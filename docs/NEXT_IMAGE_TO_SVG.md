# נקודת המשך: תמונה → SVG (הבעיה הבאה לפתור)

> מסמך זה נכתב לפני `compact` כדי לשמר את ההקשר. קרא אותו בתחילת ההמשך.

## איפה אנחנו עומדים (22/07/2026)

הצינור הנוכחי הוא **טקסט → SVG ישירות** מ־LLM אחד:
- פרומפט **מינימלי** (`src/lib/llm/prompts.ts`) — רק מה בונים, מידות, ופורמט
  ה-SVG. בלי הנחיות עיצוב/מגבלות.
- מנוע ולידציה **מתירני** — רק V1 (חוזה), V2 (חיבוריות), V3 (אי כלוא),
  V5 (פתח מינ' 0.5מ"מ). ראה `docs/REMOVED_CONSTRAINTS.md`.
- מודל נוכחי: `gpt-5.6-luna` (`OPENAI_MODEL` override; ברירת מחדל בקוד
  `src/lib/llm/openai.ts`). נפילה אוטומטית ל-Anthropic בכשל.
- **קצה גלי עובד**: `normalizeSvg` חותך cutouts לרצועה (intersection), הייצוא
  עוקב אחרי גבול המתכת האמיתי. אומת בפרודקשן — המודל מוציא רצועות גליות
  לאורך השפה העליונה/תחתונה, הצללית באמת גלית ולא מלבנית.

## הבעיה שזיהה גל

> "עם מודל 5.3 הוצאתי עיצובים יותר מוצלחים **בתמונה**. הבעיה היא להפוך תמונה
> ל-SVG בצורה מוצלחת."

כלומר: כשמבקשים מ-LLM לתאר/לצייר את הצמיד כ**תמונה** — התוצאה יפה ומוצלחת
בהרבה. צוואר הבקבוק הוא ה**המרה מתמונה ל-SVG** (וקטור נקי, סגור, שעומד בחוזה
ה-cutouts שלנו). כרגע אנחנו מדלגים על שלב התמונה ומבקשים SVG ישירות, וזה
מגביל את האיכות הוויזואלית.

## הכיוון הבא (לתכנון אחרי ה-compact)

גל יעלה **מחקר שביצע** ואז נתכנן יחד צינור בסגנון:

1. **שלב עיצוב ויזואלי** — לייצר תמונה/רנדר של הצמיד (מודל שמצייר יפה, אולי
   5.3), אולי בשליטה על סגנון/קומפוזיציה.
2. **שלב וקטוריזציה** — להמיר את התמונה ל-SVG נקי: מתאר חיצוני + חורים, נתיבים
   סגורים, ב-viewBox של הרצועה. מועמדים לבדיקה:
   - טרייסינג רסטר→וקטור (potrace / autotrace / VTracer) על מסכה ב/ל.
   - מודל multimodal שמקבל את התמונה ומחזיר SVG (image→SVG).
   - שילוב: תמונה מגדירה קומפוזיציה, מודל/טרייסר מפיק גיאומטריה, ואז המנוע
     המתירני שלנו מנקה (V1/V2/V3/V5 + לולאת תיקון).
3. **אינטגרציה** — להשחיל את הצינור החדש ב-`src/lib/llm/pipeline.ts` מבלי לשבור
   את חוזה ה-cutouts, הייצוא (DXF/SVG), וההדמיה התלת-ממדית.

## קבצים רלוונטיים להמשך

- `src/lib/llm/pipeline.ts` — תזמור יצירה + לולאת תיקון (נקודת ההשתלה).
- `src/lib/llm/prompts.ts` — הפרומפטים (system/generate/repair).
- `src/lib/llm/openai.ts` / `client.ts` / `core.ts` — ספק ה-LLM.
- `src/lib/geometry/normalize.ts` — פרסור SVG→פוליגונים, חיתוך לרצועה.
- `src/lib/geometry/validate.ts` — V1/V2/V3/V5.
- `docs/REMOVED_CONSTRAINTS.md` — מה הוסר ואיך להחזיר.
